# 🤖 LUNA

### AI Semi-Humanoid Robot (Generation I)

> **LUNA** is a semi-humanoid AI robot developed by **TechBots-SIT** that combines robotics, embedded systems, artificial intelligence, computer vision, and natural language interaction into a single interactive platform. Built around the Raspberry Pi and Arduino Mega, LUNA was designed to demonstrate autonomous interaction, mobility, and intelligent human-robot communication.

---

## 📖 Overview

LUNA is the first-generation semi-humanoid robot developed by TechBots as an AI-powered robotic assistant. The robot integrates Large Language Models (LLMs), Computer Vision (CV), voice interaction, and embedded motor control to create an interactive robotic platform.

The system combines a Raspberry Pi for AI processing with an Arduino Mega for low-level motion control, allowing intelligent decision-making while maintaining reliable real-time hardware control.

## 🏗 System Architecture

The robot is divided into two independent processing units.

### Raspberry Pi

Responsible for:

* Artificial Intelligence
* Large Language Model integration
* Computer Vision
* Voice Processing
* Object Recognition
* Digital Eye Display

### Arduino Mega

Responsible for:

* Robot locomotion
* Motor control
* Gesture control
* Servo actuation
* Communication with motor drivers
